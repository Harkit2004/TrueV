Set up the see through on droplet:
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
bash Miniconda3-latest-Linux-x86_64.sh
source ~/.bashrc
git clone https://github.com/shitagaki-lab/see-through
cd see-through
conda create -n see_through python=3.12 -y
conda activate see_through
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/rocm7.2
pip install -r requirements.txt
ln -sf common/assets assets
cd ~
git clone 



On the droplet:

  A) Clone https://github.com/shitagaki-lab/see-through and follow its README (conda env, PyTorch CUDA wheel, pip install -r requirements.txt, ln -sf common/assets assets from repo root).

  B) Clone this hackathon repo if needed; cd backend; use .venv and pip install -r requirements.txt.

  C) In backend/.env on the droplet set:

       SEE_THROUGH_REPO=/path/to/see-through

       SEE_THROUGH_PYTHON=/path/to/that/conda/env/bin/python

     Optional shared auth (recommended if the worker port is ever exposed beyond localhost):

       SEE_THROUGH_SERVICE_SECRET=your-long-random-string

  D) Start the worker (listens on loopback only by default):

       python run_see_through_worker.py

     Defaults: SEE_THROUGH_WORKER_HOST=127.0.0.1, SEE_THROUGH_WORKER_PORT=9105.

     Optional: HF_TOKEN on the droplet for faster Hugging Face downloads when See-through pulls models.



On your laptop (main FastAPI):

  E) Open SSH local forwards alongside your Comfy tunnel, for example:

       ssh -N -L 8188:127.0.0.1:8188 -L 9105:127.0.0.1:9105 USER@DROPLET_IP

     Adjust ports if your droplet uses different Comfy or worker ports.

  F) In backend/.env on the laptop set:

       COMFYUI_BASE_URL=http://127.0.0.1:8188

       SEE_THROUGH_SERVICE_URL=http://127.0.0.1:9105

       SEE_THROUGH_SERVICE_SECRET=same-as-droplet-if-you-set-it

     You do not need SEE_THROUGH_REPO on the laptop when SEE_THROUGH_SERVICE_URL is set.

  G) python run.py on the laptop; POST /api/see-through/decompose uploads the image here and the backend forwards it to the droplet worker over the tunnel.
